# 🔥 Wildfire Tracker

A real-time wildfire tracking application built with React that displays active wildfire events around the world using NASA's EONET API and an interactive map.

## Features

- 🗺️ Interactive world map showing real-time wildfire locations
- 🛰️ Powered by NASA EONET API for live event data
- 📍 Click on any marker to see wildfire details
- 🌍 6000+ active events tracked globally

## Tech Stack

- React
- Leaflet / React-Leaflet (interactive maps)
- NASA EONET API (real-time wildfire data)
- OpenStreetMap

## Getting Started

```bash
npm install --legacy-peer-deps
npm start
```

## API

This project uses the NASA EONET API v3:
https://eonet.gsfc.nasa.gov/api/v3/events

No API key required — completely free and open!
